这套指南把 vishine 主题的每个特性都拆成一篇可动手的教程。不堆术语，每篇先讲「为什么」，再给「怎么做」，配上可直接复制的配置和命令。

{{< lead >}}
想自定义就自定义，不想管就全自动 —— vishine 在 Blowfish 的基础上，把中文技术博客最常用的能力做成了零配置默认值。
{{< /lead >}}

## 各篇导览

- **[安装与快速开始](installation/)** —— clone 主题、最小 `hugo.toml`、`hugo server` 跑起来。
- **[配色与板块色彩编码](color-schemes/)** —— 三套 scheme（paper / clean / dark）+ 五大板块色映射。
- **[自动封面生成器](auto-cover/)** —— 不配图也有统一风格封面，`params.cover` 全选项详解。
- **[Shortcodes 大全](shortcodes/)** —— badge / callout / lead / timeline 的用法与实时渲染效果。
- **[Mermaid 图表](diagrams/)** —— 用围栏画流程图，随配色自动翻色。
- **[菜单与多级下拉](menus/)** —— 顶栏一级菜单与二级下拉的配置方法。
- **[部署到 GitHub Pages](deployment/)** —— GitHub Actions 自动构建发布，含完整 workflow。

{{< callout type="tip" >}}
建议按顺序读：先跑起来（安装），再调外观（配色 / 封面），最后写内容（shortcodes / 图表）和上线（部署）。
{{< /callout >}}
